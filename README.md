# XCalibYieldAI

This repository contains the full modular implementation of the SCI paper 'XCalibYieldAI: A Dual-Stage Explainable and Calibrated Deep Learning Framework for Crop Yield Forecasting'.

## Directory Structure
- `data/`: Preprocessing pipeline
- `models/`: Model definition and architecture
- `train/`: Training loop
- `evaluate/`: Evaluation scripts
- `inference/`: Inference and deployment
- `utils/`: Logger and visualization helpers
- `configs/`: Configuration files
- `logs/`: Logs will be stored here
- `notebooks/`: Jupyter notebooks for analysis
- `outputs/`: Prediction outputs and results
